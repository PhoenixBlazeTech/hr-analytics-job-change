## Objective
Analyze factors associated with job change decisions using HR data.

## Dataset
HR Analytics: Job Change of Data Scientists (Kaggle)

## Tools
- Excel (Power Query, Pivot Tables)
- Power BI
- GitHub

## Process
1. Data cleaning and transformation in Excel using Power Query
2. Feature engineering (experience, salary bands)
3. KPI definition and exploratory analysis
4. Interactive dashboard built in Power BI

## Key Insights
- Higher job change probability among low-experience profiles
- Certain company types show higher turnover
- Education level correlates with job stability

## Deliverables
- Clean dataset (Excel)
- Power BI dashboard
- Documentation and visuals
